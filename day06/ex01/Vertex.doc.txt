<- Vertex ----------------------------------------------------------------------
An instance is contructed from XYZ values:
new Vertex( array( 'x' => 10, 'y' => 10, 'z' => 4 ) );

W coordinate and a color are optional
new Vertex( array( 'x' => 10, 'y' => 10, 'z' => 4, 'w' => 3, 'color' => new Color('red' => 0, 'green' => 255, 'blue' => 0') ) );
